# Exercise 4: News Screen Tabs

## Part 1 - Module 4: Tab Navigation

### Goal
Create a news screen with three tabs: Sports, Technology, and Business.

### Requirements
1. Wrap the news screen in a `DefaultTabController` with length: 3.
2. Add a `TabBar` (inside the AppBar's bottom) with tabs labeled Sports, Technology, and Business.
3. Add a `TabBarView` with three corresponding pages, each showing placeholder content for that category.
4. Confirm you can switch categories both by tapping a tab and by swiping horizontally.

### Hint
DefaultTabController's length must match both the number of Tab widgets in TabBar and the number of children in TabBarView.

### Concepts Covered
- DefaultTabController
- TabBar
- TabBarView
- Customizing tabs
- Tab navigation vs Bottom Navigation
