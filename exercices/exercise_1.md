# Exercise 1: Two-Screen Navigation

## Part 1 - Module 1: Introduction to Navigation

### Goal
Create two screens and navigate between them using a button.

### Requirements
1. Create a `FirstScreen` with a button labeled 'Go to Second Screen'.
2. Create a `SecondScreen` with a button labeled 'Go Back'.
3. On FirstScreen's button press, use `Navigator.push()` with a `MaterialPageRoute` to open SecondScreen.
4. On SecondScreen's button press, use `Navigator.pop()` to return to FirstScreen.

### Hint
Wrap each screen in its own Scaffold so each has its own AppBar/body.

### Concepts Covered
- What is navigation?
- Navigation stack
- Routes and screens
- Navigator
- MaterialPageRoute
